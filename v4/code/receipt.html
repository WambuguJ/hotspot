Section details:

This section displays receipt details, including:.

  • receipt.ref (reference number)
  • receipt.date (date of transaction)
  • receipt.amount (total amount spent)
  • receipt.vat (tax applied)
  • receipt.description (category, e.g., "kitchen")

These details are gotten from the selected receipt.

The highlighted entries corresponds to the highligted image, "333" file path and consumer's details from the top left section.

The fields in this section can be edited and saved
